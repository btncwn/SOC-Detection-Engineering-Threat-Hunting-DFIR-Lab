# SOC Detection Engineering, Threat Hunting & DFIR Lab

This repository documents an end-to-end SOC lab designed to simulate a realistic attack lifecycle and blue-team investigation workflow.

The lab combines vulnerability assessment, attack simulation, endpoint telemetry, SIEM monitoring, threat hunting, threat intelligence enrichment, DFIR, MITRE ATT&CK mapping, and incident reporting.

## Lab Flow

1. Vulnerability Assessment with Nessus  
2. Attack Simulation with Kali Linux  
3. Endpoint Telemetry Collection with Sysmon  
4. Log Ingestion with Splunk Universal Forwarder  
5. Threat Hunting in Splunk  
6. SOC Dashboard Monitoring  
7. Threat Intelligence Enrichment with MISP  
8. DFIR Investigation with Velociraptor  
9. MITRE ATT&CK Mapping  
10. Incident Reporting

## Project Goal

The goal of this lab is to demonstrate practical SOC skills including detection engineering, vulnerability management, threat hunting, incident response, and security investigation using real telemetry from a controlled lab environment.
