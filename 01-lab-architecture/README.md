# 01 - Lab Architecture

Design and implementation of a complete Security Operations Center (SOC) lab environment used for Detection Engineering, Threat Hunting, Incident Response, Threat Intelligence, and SOAR Automation.

The lab integrates Windows and Linux systems, centralized log management, endpoint telemetry, vulnerability assessment, threat intelligence platforms, and security automation workflows.

Projects include:

* Splunk Enterprise Deployment
* Windows 7 Endpoint Configuration
* Windows 11 Security Testing Environment
* Kali Linux Attack Platform
* Sysmon Telemetry Collection
* Splunk Universal Forwarder Configuration
* Nessus Vulnerability Assessment Platform
* MISP Threat Intelligence Platform
* Docker and Docker Compose Deployment
* Network Architecture and Connectivity
* Log Collection and Centralization
* Security Monitoring Infrastructure

Key Outcomes:

* Centralized Security Monitoring
* Endpoint Visibility
* Threat Intelligence Integration
* Detection Engineering Capability
* Incident Response Readiness
* Security Automation Foundation
